# Memory Game

Simple web-based memory matching game.

## Run
1. Open `index.html` in a browser.
2. Click cards to flip and match pairs.
3. Use `Restart` to start new round.

## Game rules
- 16 cards, 8 pairs.
- Click two cards each turn.
- If pair matches, they stay revealed.
- If not, cards flip back after a short delay.
- Count of attempts and matches shown.
